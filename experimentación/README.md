# Descripción del Directorio

Este directorio contiene una carpeta denominada **"imagenes"** que alberga todas las ejecuciones posibles con dos y tres centros de las imágenes de prueba de la aplicación.

## Estructura de la Carpeta "imagenes"

La carpeta **"imagenes"** contiene imágenes con los nombres formateados de la siguiente manera:

fechaDeEjecucion_1__nombreDeLaImagen__algoritmoInvariante.extension
fechaDeEjecucion_2__nombreDeLaImagen__Original_algoritmoAgrupamiento_numeroDeCentros_c_acierto_%.extension
fechaDeEjecucion_3__nombreDeLaImagen__algoritmoInvariante_algoritmoAgrupamiento_numeroDeCentros_c_acierto_%.extension

## Uso de la Memoria Caché

Estos son los mismos nombres que acepta la memoria caché, por lo que se pueden copiar y pegar imágenes en la memoria caché para agilizar la ejecución.

## Archivos Excel

Además, hay dos archivos Excel que resumen el acierto de cada ejecución, proporcionando una forma más visual de revisar los resultados:

- **image_percentages_2centros.xlsx**
- **image_percentages_3centros.xlsx**

Estos archivos permiten una revisión rápida y visual de los resultados de las ejecuciones, facilitando la comparación y el análisis de los datos obtenidos.
