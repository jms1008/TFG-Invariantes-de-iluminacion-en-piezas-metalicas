Fuzzy C-Means

References

Fuzzy C-Means (FCM) is a fuzzy logic-based clustering algorithm that allows a data point to belong to more than one group (or cluster) with different degrees of membership. Unlike K-Means, where each point belongs exclusively to a single cluster, in FCM each point has a membership value for each cluster, which indicates the degree to which a point belongs to a specific cluster.

Why use it?

Fuzzy C-Means is especially useful in applications where the boundaries between classes are not clearly defined, which commonly occurs in image segmentation problems. Thus, segmenting an image that has overlapping regions with different degrees of membership will give a better result, which is useful for segmenting areas with smooth transitions.
